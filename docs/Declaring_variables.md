# Declaring variables
## Unprotected variables
### String
```
let a = "Hi";   // String
```
### Integer
```
let b = 123;    // Integer
```
### Float
```
let c = 1.23    // Float
```
### Boolean
```
let d = true    // Boolean
```

## Protected variables
```
let protected a = "Hi";
let a = 123;    // Error
```
